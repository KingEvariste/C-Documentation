Getting Started with C

Steps in getting started

  1. Download C Compiler
  2. Install the C Compiler
  3. Get an Integrated Development Environment (IDE)
  4. Start a project within the IDE
  5. Code up"Hello World" and run the code

For a Compiler it is recommended to use GCC

For IDE it is recommended to use VSCode

C Entry Point

The entry point to a C application is the main function.

It is categorized byt its int argc and char *argv[] as parameters to the function 

        
          int main(int argc, char *argv[]){
            
          }
Printing to the console

In order to print to the console. We use the printf function from the stdio library

        
          #include stdio.h
          
          int main(int argc, char *argv[]){
            printf("Hello World");
          }

In this example we are printing out "Hello World" to the console when we compile and run the program.

Declaring Functions

Functions are declared in C first by declaring the return type and then the function name after that are the function parameters enclosed in parentheses and separated by commas.

Function with no return type and just prints "Hello World" 

        
          void printHelloWorld(){
            printf("Hello World");
          }

Function with integer return type and adds the two parameters x and y 

        
          int addNum(int x, int y){
            return x + y;
          }
Structs

Structures (also called structs) are a way to group several related variables into one place.

Each variable in the structure is known as a member of the structure.

Unlike an array, a structure can contain many different data types (int, float, char, etc.).

        
          struct MyStructure {   // Structure declaration
            int myNum;           // Member (int variable)
            char myLetter;       // Member (char variable)
          }; // End the structure with a semicolon
